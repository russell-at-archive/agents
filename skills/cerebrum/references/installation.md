# Cerebrum Installation

## Prerequisites

Cerebrum is an Obsidian vault. It relies on the Obsidian app and CLI.

1.  **Install Obsidian:** [https://obsidian.md/download](https://obsidian.md/download)
2.  **Register CLI:**
    -   Open Obsidian
    -   Go to **Settings → General → Command line interface → Register CLI**
3.  **Ensure "cerebrum" vault exists:**
    -   If not, create it in Obsidian.
    -   Run `obsidian vaults` to verify it appears in the list.

For full CLI setup instructions, see:
`skills/using-obsidian-cli/references/installation.md`
